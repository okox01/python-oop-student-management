# 🎓 Python OOP Student Management System

A console-based Student Management System built using **Object-Oriented Programming (OOP)** principles in Python.

This project demonstrates clean code structure, inheritance, encapsulation, and method overriding in a real-world mini application.

---

## 📌 Project Overview

This system allows users to:

- Add regular students
- Add graduate students (with thesis title)
- View all students
- Calculate average grade
- Find highest grade

The application runs in a simple interactive menu in the terminal.

---

## 🧠 OOP Concepts Implemented

This project demonstrates:

- ✅ Classes and Objects
- ✅ Constructor (`__init__`)
- ✅ Instance Methods
- ✅ Encapsulation (Private attributes using `__grades`)
- ✅ Getter Methods
- ✅ Inheritance (`Graduate` class inherits from `Student`)
- ✅ Method Overriding
- ✅ `super()` function usage
- ✅ List of objects (Object collection handling)
